# Reflection

## What patterns or surprises did you notice?

I noticed that each function performs one specific task. Keeping functions small and focused makes the code easier to understand and be reused. 

## What logic challenged your thinking?

The isEven() and isOdd() functions required using the modulo (%) operator to determine whether a number is divisible by 2.

## How might this kind of toolkit be used in the real world?

A collection of utility functions like these can be reused in larger programs to perform common mathematical operations without rewriting the same code multiple times thus being time efficient.
