# Predictions

## double()
I predict that the double() function will return double the value of the number passed into it. 

Example:
double(5) should return 10.

## square()
I predict that the square() will return the number multiplied by itself. 

Example:
square(4) should return 16. 

## isEven()
I predict that the isEven() function will return true if the number is even and false if the number is odd. 

Example:
isEven(8) should return true. 

## isOdd()
I predict that the isOdd() function will return true if the number is odd and false if the number is even. 

Example:
isOdd(7) should return true.
